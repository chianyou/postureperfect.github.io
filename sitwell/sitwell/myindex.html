Motivation

改善長期久坐所產生的問題

現代生活中，長時間的辦公、學習以及娛樂活動，都使人們長時間保持坐姿，這對我們的健康和生活品質造成了負面影響。不正確的坐姿可能引發許多健康問題，包括背部疼痛、頸部疼痛、姿勢問題，甚至對內部器官造成壓力。我們的製作動機是改善人們的坐姿，提高他們的生活品質和健康。我們計劃開發一個使用鏡頭來檢測人們坐姿是否正確的應用程式，並提供坐姿矯正的建議。

功能介紹

坐姿檢測

使用內置或外部鏡頭捕捉您的坐姿，並即時評估其正確性。應用程式將分析您的坐姿，提供坐姿評分以及改進建議

Read More
即時警示

應用程式會立即警示您，當偵測到不正確的坐姿。這將幫助您在體驗不舒適或疼痛之前即時調整姿勢。

Read More
坐姿歷史追蹤

記錄您的坐姿歷史，以便您可以追蹤和比較不同時間段的坐姿表現。這有助於了解坐姿改進的進展

Read More
矯正建議

提供專業的坐姿改進建議，包括如何調整坐姿，何時應該休息，並提供矯正運動或伸展建議。

Read More
目標設定

設定個人坐姿改進目標，並追蹤您的進度。這有助於提醒您始終保持正確的坐姿。

Read More
統計和報告

提供關於坐姿狀況的詳細統計數據和報告，以便您可以深入瞭解您的坐姿習慣。
